# InstaWeb QA Assignment

## Project Overview
This project contains QA testing for:
https://websites.co.in

It includes manual testing, basic automation, and bug reporting.

---

## What is included

- Excel file (Manual + Automation + Bugs)
- Word file (Priority Judgment)
- Automation scripts (Selenium)
- Screenshots and Screen recording

---

## Testing done

- Registration testing
- Login testing
- Form validation
- Basic UI checks
- Navigation testing

---

## Bugs found

- No success message after registration
- No success message after login
- Phone number allows more than 10 digits
- UI alignment issues
- Navigation issue

---
## How to run automation scripts

1. Install Selenium:  pip install selenium
2. Run any script:python open_login.py
